# 🏙️ Portal de Serviços Municipais - Conceito & Protótipo

> **Proposta de modernização para portais governamentais, focada em UX e acessibilidade.**

## 💡 O Projeto
Estudo de caso e template de interface para aplicativos municipais, utilizando identidade visual de cidade fictícia.

## 🎨 Design
- **Interface Limpa:** Foco na clareza e transparência das informações.
- **Navegação Intuitiva:** Menu inferior para acesso rápido aos serviços.
- **Tipografia:** Uso da fonte *Poppins* para leitura confortável no mobile.
- **Link do Projeto:** [Acesse o Figma aqui](https://www.figma.com/design/1KES6TEWLE73BJsM7gxBqs/Projetos-de-Portais?node-id=0-1&t=TRTgzsN0o18xP8b4-1)

## 🏗️ O que tem no Protótipo?
Sugestões de funcionalidades:
- ✅ **Home:** Painel dinâmico com carrossel de notícias.
- ✅ **Serviços:** Guia organizado por categorias (Saúde, Educação, etc).
- ✅ **Ouvidoria:** Canal direto e simplificado para o cidadão.
- ✅ **Notícias:** Feed oficial da cidade.

## 🛠️ Tecnologias
- **Flutter** & **Dart**
- **Font Awesome Icons**
